# Football Matches App
App for getting today football matches and web page with sport news

## Technologies & Libraries
Project is created with:
* Android Studio (Kotlin)
* Room
* Retrofit
* MVVM
* Clean Architecture
* LiveData
* Coroutines
* Glide
* Dagger 2
* OneSignal
